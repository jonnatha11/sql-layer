There are three software requirements that must be met before the FoundationDB SQL Layer can be installed.

Requirements
  • OS X version 10.7 or later
  • FoundationDB Client library version 3.0.0 or later
  • Java Runtime Environment version 7 or later


FoundationDB Client
This client library can be found at foundationdb.com/get#osx. Installing FoundationDB Server is also recommended.

Important: The SQL Layer cannot be installed without this client present.

Java Runtime Environment
The Java runtime can be found at Java SE Downloads. If you are unsure of which to install, choose JRE 7 and select the dmg.
