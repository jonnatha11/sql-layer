delete-simple: delete of range given by indexed column

delete-subquery: delete from table which is used in predicate subquery

insert-other-table: insert via select from another table

insert-same-table: insert via select from same table

insert-values: insert literals only

update-other-index: update indexed columns with predicate on different columns

update-pk: update pk of all rows

update-same-index: update indexed columns with predicate on same columns

update-single-pk: update pk of a single row

update-single-pk-child-1: update pk of a single child row, parent unspecified

update-single-pk-child-2: update pk of a single child row, parent specified
